Por Coruña for the Nokia N9

What is it?

Por Coruña is a public transportation application for the Nokia N9. It currently provides information about A Coruña's bus lines, bus stops and bus arrival times, allowing users to save their favorite bus stops.

How does it work?

Por Coruña is happily using Móvete's API. Móvete has been developed by Víctor Pena, who has also provided an iOS implementation, already available in the App Store.

Bugs & Feedback

Por Coruña has been developed with the hope of being useful, but for sure things can be improved. If you find bugs or have any ideas to make it better, don't hesitate to create an entry in our bug tracker.
